# 물주기 알림_lite 개인정보 처리방침

**최종 업데이트: 2025년 11월 9일**

## 1. 수집하는 개인정보

물주기 알림_lite는 다음과 같은 정보를 수집하지 않습니다:
- 개인 식별 정보 (이름, 이메일, 전화번호 등)
- 위치 정보
- 기기 고유 식별자

본 앱은 **오프라인으로 작동**하며, 모든 데이터는 사용자의 기기에만 저장됩니다.

## 2. 데이터 저장

### 로컬 저장
- **식물 정보**: 식물 이름, 사진, 물주기 주기, 마지막 물 준 날짜
- **알림 설정**: 알림 시간 및 활성화 상태
- **앱 설정**: 사용자 환경 설정

모든 데이터는 **기기 내부 저장소**에만 보관되며, 외부 서버로 전송되지 않습니다.

### 데이터 백업
- 사용자가 직접 JSON 파일로 내보내기 가능
- 백업 파일은 사용자가 선택한 위치에만 저장

## 3. 제3자 서비스

### Google AdMob
본 앱은 광고 제공을 위해 Google AdMob을 사용합니다. AdMob은 광고 맞춤 설정을 위해 다음 정보를 수집할 수 있습니다:
- 광고 ID
- IP 주소
- 기기 정보 (모델, OS 버전)
- 앱 사용 통계

**AdMob 개인정보 처리방침**: https://policies.google.com/privacy

**광고 맞춤 설정 해제 방법**:
- Android: 설정 > Google > 광고 > 광고 맞춤설정 선택 해제
- iOS: 설정 > 개인정보 보호 > 광고 > 광고 추적 제한

## 4. 권한 사용

### Android 권한
- **알림 (POST_NOTIFICATIONS)**: 식물 물주기 알림 전송
- **정확한 알람 (SCHEDULE_EXACT_ALARM)**: 정확한 시간에 알림 전송
- **부팅 완료 수신 (RECEIVE_BOOT_COMPLETED)**: 기기 재시작 후 알림 재설정
- **배터리 최적화 무시 (REQUEST_IGNORE_BATTERY_OPTIMIZATIONS)**: 알림 신뢰성 향상
- **인터넷 (INTERNET)**: 광고 표시

### iOS 권한
- **알림**: 식물 물주기 알림 전송
- **사진 라이브러리**: 식물 사진 선택 (선택사항)
- **카메라**: 식물 사진 촬영 (선택사항)

모든 권한은 앱의 핵심 기능을 위해서만 사용됩니다.

## 5. 데이터 보안

- 모든 데이터는 기기 내부에만 저장
- 외부 서버로 전송되지 않음
- 네트워크 통신은 광고 표시에만 사용

## 6. 어린이 보호

본 앱은 13세 미만 어린이를 대상으로 하지 않으며, 고의로 어린이의 개인정보를 수집하지 않습니다.

## 7. 데이터 삭제

- 앱 삭제 시 모든 데이터가 함께 삭제됩니다
- 설정 메뉴에서 "전체 데이터 삭제" 가능

## 8. 개인정보 처리방침 변경

본 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우 앱 내 공지사항으로 알려드립니다.
