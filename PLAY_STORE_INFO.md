**전체 설명** (최대 4000자):
🌱 물주기 알림_lite - 식물 관리가 쉬워집니다!

식물을 사랑하지만 물주는 날짜를 자주 잊으시나요?
물주기 알림_lite가 여러분의 식물 친구들을 건강하게 지켜드립니다!

## 주요 기능

### 📅 맞춤형 물주기 알림
- 각 식물마다 다른 물주기 주기 설정
- 원하는 시간에 정확한 알림
- D-day 카운터로 한눈에 확인

### 🌿 간편한 식물 관리
- 식물 사진 추가로 시각적 관리
- 마지막 물 준 날짜 자동 기록
- "물 줬어요" 버튼으로 간편 업데이트

### 💾 데이터 백업 & 복구
- JSON 파일로 내보내기
- 기기 변경 시 쉽게 데이터 이전
- 모든 데이터는 기기에만 저장 (개인정보 안전)

### 🎨 깔끔한 디자인
- 직관적인 사용자 인터페이스
- Material 3 디자인
- 라이트 모드 지원

## 권한 안내

- **알림**: 물주기 시간 알림
- **정확한 알람**: 정확한 시간에 알림 전송
- **사진 라이브러리**: 식물 사진 추가 (선택)
- **인터넷**: 광고 표시

## 개인정보 보호

물주기 알림_lite는 사용자의 개인정보를 수집하지 않습니다.
모든 데이터는 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다.

자세한 내용은 개인정보 처리방침을 참조해주세요.
