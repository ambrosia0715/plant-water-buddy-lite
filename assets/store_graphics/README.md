# Play Store 그래픽 에셋 가이드

## ✅ 준비 완료

### 1. 앱 아이콘 (512x512)
**파일**: `assets/store_graphics/app_icon_512.png`
- ✅ 크기: 512 x 512 픽셀
- ✅ 형식: 32비트 PNG (알파 채널 포함)

### 2. Feature Graphic (1024x500)
**파일**: `assets/store_graphics/feature_graphic.png`
- ✅ 크기: 1024 x 500 픽셀
- ✅ 형식: PNG 또는 JPG
- ✅ 디자인: 앱 이름 + 슬로건 + 그라데이션 배경

---

## ⏳ 추가 필요 (스크린샷)

### 3. 스크린샷 (최소 2장, 최대 8장)

**요구사항:**
- 크기: 320~3840 픽셀 (세로 또는 가로)
- 형식: PNG 또는 JPG
- 비율: 16:9 또는 9:16 권장

**권장 스크린샷:**

1. **홈 화면** (식물 목록)
   - D-day 카운터가 보이는 메인 화면
   - "오늘 할 일" 섹션

2. **식물 추가 화면**
   - 식물 추가 양식
   - 사진, 이름, 주기 설정

3. **식물 상세 화면**
   - 식물 정보
   - "물 줬어요" 버튼

4. **알림 화면** (선택)
   - 알림이 울리는 모습 (스크린샷 찍기 어려움)

5. **설정 화면** (선택)
   - 데이터 백업/복구 옵션

### 스크린샷 촬영 방법

#### 방법 1: Android 에뮬레이터/실기기
```bash
# 앱 실행
flutter run

# Android Studio에서 스크린샷 캡처
# 또는 기기의 스크린샷 기능 사용
```

#### 방법 2: 수동 캡처
1. 앱을 실행합니다
2. 각 화면으로 이동합니다
3. 스크린샷을 찍습니다 (Android: Power + Volume Down)
4. 파일을 `assets/store_graphics/screenshots/` 폴더에 저장
